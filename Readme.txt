Recipe CRUD API :-

 => A RESTful CRUD API for managing recipes built using Node.js, Express.js, and MongoDB (Mongoose) following the MVC architecture pattern.


Tech Stack:-

 => Node.js
 => Express.js
 => MongoDB
 => Mongoose
 => Postman


Features :-

 => Create a new recipe
 =>Get all recipes
 => Get recipe by ID
 => Update recipe
 => Delete recipe
 => Proper error handling & validation
